websites from AI in a 6-stage workflow.

Core Functionality:
✅ AI-powered PRD generation (Claude 3.5 Sonnet)

APPLICATION FEATURES
================================================================================

Landing Page:
  - Hero section with value proposition
  - Feature cards highlighting capabilities
  - Clear call-to-action buttons
  - Professional, responsive design
  - No AI-ish aesthetic

Builder Interface (6 Stages):
  1. Prompt Input - User describes website idea
  2. PRD Generation - AI creates comprehensive requirements
  3. Frontend Code - AI generates HTML/CSS/JS with live preview
  4. Backend Code - AI generates API code and database schema
  5. Integration Testing - Automated compatibility tests
  6. Download - Complete project as ZIP file

Live Preview System:
  - Real-time HTML rendering
  - Sandbox iframe for security
  - Interactive component support
  - Responsive preview

Dashboard:
  - Project listing
  - Download history
  - Project management (ready for enhancement)

API ENDPOINTS
================================================================================

Fully Implemented & Working:

POST /api/ai/generate-prd
  Input: { prompt }
  Output: { prd: "..." }
  AI Model: Claude 3.5 Sonnet
  Time: 10-30 seconds

POST /api/ai/generate-frontend
  Input: { prompt, prd }
  Output: { code: "..." }
  AI Model: GPT-4o
  Time: 15-45 seconds

POST /api/ai/generate-backend
  Input: { prompt, prd, frontendCode }
  Output: { code: "..." }
  AI Model: GPT-4o
  Time: 15-45 seconds

POST /api/projects/download
  Input: { prompt, prd, frontendCode, backendCode }
  Output: { downloadUrl, fileName }
  Storage: Vercel Blob
